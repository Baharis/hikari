
This file contains a list of names for available point groups in Kesshou.
All point group names base on the short Hermann-Maugin notation,
but minor divergences had to be made in order to consider the limitations
of python language, available namespace and alternative choices of axes.
This resulted in a "Internal notation", both human- and python- readable.
Internal notation names always start from PG, which stands for Point Group,
followed by a name of the point group inside square brackets and single quotes.

Abbreviations used in the table:
Hermann-Maugin notation - HMNot
Schoenflies notation    - SNot
Internal notation       - INot
Asterisk (*) denotes alternative choice of axes
